# Prompt Pack

Aria v2 redesign prompts, ordered for execution:

1. `01_arc2_task_sketch_cards.md`
2. `02_sketch_ir.md`
3. `03_decomposition_layer.md`
4. `04_sketch_fit_two_families.md`
5. `05_sketch_compiler.md`
6. `06_sketch_oriented_refinement.md`
7. `07_learned_proposal_layer.md`

These prompts are meant to move Aria from fixed rule-family matching toward:

- task-local sketch synthesis
- shared decomposition primitives
- sketch compilation to executable programs
- optional learned proposal with exact symbolic verification

They are also meant to align explicitly with ARC-AGI-2's published stressors:

- symbolic interpretation
- compositional reasoning
- contextual rule application
- efficiency over brute-force search
